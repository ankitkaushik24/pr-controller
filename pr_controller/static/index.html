Slack Integration

Configure a Slack Workflow Builder webhook so you can message reviewers directly from the dashboard.

How to create a webhook in Slack
  1. Open Slack → Automations → New Workflow.
  2. Choose From a webhook as the trigger.
  3. Add these variables (all Text type): user_email, message, pr_number, pr_title, pr_url, event_type.
  4. Add a Send a message to a person action using {{user_email}} as the recipient and {{message}} as the body. Optionally append {{pr_url}}.
  5. Publish the workflow and copy the Webhook URL here.
